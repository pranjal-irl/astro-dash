

<h1 align="center">AstroDash</h1>

<p align="center">An endless space survival game where you have to dodge the asteroids coming towards you and each dodged asteroid equals 1 point. The difficulty increases with time with a max speed cap I added for this level 1 version of game. </p>

## Navigation
- W/↑ : Up
- A/← : Left
- S/↓ : Down
- D/→ : Right


## Tech Stack
- Godot
- GDScript


## Running Locally

1) Clone the repository:

```sh
 git clone https://github.com/pranjal-irl/astro-dash.git
```

2) Open Godot Game Engine

3) Click "Import" and import the cloned file
